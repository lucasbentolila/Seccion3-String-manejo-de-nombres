Programa Manejo de Nombres

Ejercicio práctico del curso de Java que consiste en solicitar tres nombres y generar uno nuevo para cada nombre tomando:

El segundo carácter en mayúscula

Un punto

Los dos últimos caracteres del nombre

Finalmente, se imprime un resultado combinando los tres nuevos valores separados por un guion bajo.

✔ Tecnologías

Java

✔ Contenido utilizado

Scanner

charAt()

substring()

toUpperCase()
